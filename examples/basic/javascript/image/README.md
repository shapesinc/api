# Images example for the Shapes API

This example demonstrated how to send an image to your shape.

## Prerequisites

- Node.js 18+
- npm

## Setup

```bash
npm install
```

This will install the required dependencies.

## Usage

Copy the `.env.example` file to `.env` and add your API keys.

```bash
node main.js
```

Alternatively, using npm scripts:

```bash
npm start
```
